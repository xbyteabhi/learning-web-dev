SEO & Core Web Vitals

SEO:
SEO stands for SEARCH ENGIENE OPTIMISATION, it helps websites to stand out among the other websites on the browser and ranks in the top list.

Core Web Vitals

Sometimes your content quality is preffered more than core web vitals, but when your website is competing with website having same quality content, then at this situation Core Web Vitals matters more for ranking of the site

CLS
Cumulative Layout Shift (CLS) means how much is your site page shifting while loading this directly impacts user experience, to handle this adding height and width to images is preffered
LCP
Largest Conxetual Paint (LCP) means how much time is taken by the largest image (or the important part of the web) to load in the site, as per the google 2.5 seconds is preffered & good time for a website, more than this is bad for page health
FID
First Input Delay states how much time should be taken by site to start processing the input given by the user, as per google 100 miliseconds or less is preffered for a good website
