Inline CSS
Inline CSS is added directly in the element using a style attribute, but using this way we have to write css for every element particulary which is definately not a good way
Internal CSS
Internal CSS is added in the head tag below the title tag using a styule tag, and in this we can target multiple elements and add styling to all of them at a time, but when we write long codes it becomes some sort of messy and a bit hard to manage html and css at the same time
External CSS
External CSS is a good way to write a css for long sites, its make it easy to manage the files
